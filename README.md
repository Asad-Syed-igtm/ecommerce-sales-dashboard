# E-commerce Sales Dashboard

## Overview
This project is a Power BI dashboard built to analyze e-commerce sales data. It provides insights into product performance, category trends, and overall sales activity.

## Features
- Total sales and quantity KPIs
- Top 5 products by revenue
- Sales by category
- Sales trends over time
- Interactive filtering by category

## Tools Used
- Power BI
- Excel / CSV data

## Project Files
- `sales_dashboard.pbix` – Power BI report
- `ecommerce_sales_dataset.csv` – dataset used
- Dashboard screenshot included
